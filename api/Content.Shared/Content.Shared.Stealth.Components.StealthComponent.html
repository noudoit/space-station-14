Class StealthComponent

Add this component to an entity that you want to be cloaked. It overlays a shader on the entity to give them an invisibility cloaked effect. It also turns the entity invisible. Use other components (like StealthOnMove) to modify this component's visibility based on certain conditions.

LastVisibility

Last set level of visibility. The visual effect ranges from 1 (fully visible) and -1 (fully hidden). Values outside of this range simply act as a buffer for the visual effect (i.e., a delay before turning invisible). To get the actual current visibility, use GetVisibility(EntityUid, StealthComponent) If you don't have anything else updating the stealth, this will just stay at a constant value, which can be useful.
